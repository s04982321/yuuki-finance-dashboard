# 家計簿アプリ

PythonとStreamlitを使って作成したシンプルな家計簿アプリです。

## 機能
・収入と支出の記録  
・カテゴリ別支出ランキング  
・支出の円グラフ可視化  
・月別支出グラフ  
・貯蓄率の自動計算  
・CSVダウンロード  

## 使用技術
- Python
- Streamlit
- Pandas
- Plotly
